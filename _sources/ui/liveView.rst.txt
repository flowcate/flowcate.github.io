
Live View
+++++++++

Live View fetches the event triggers when a trackable is detected in a zone or fence.
An event is triggered when a trackable enters a fence, as well as when a trackable exits a fence.
Additionally, a timestamp is displayed in the left side pane within the user interface.

.. image:: images/hub_i_live.png
    :class: screenshot

When a trackable enters a fence, the fence is highlighted in green.
Similarly, when a trackable leaves a fence, it is highlighted in red.
